Principais formatações

Texto marcado

Este é um exemplo de texto marcado usando a tag <MARK>.

Texto grande e pequeno

Lorem ipsum dolor sit amet consectetur adipisicing elit. Beatae, praesentium eaque aut dicta molestiae sed, possimus veniam ea neque corporis iste ad saepe dolor autem ducimus. Iure unde beatae dolor.
Usando as TAGS <BIG> (desuso) e <SMALL>.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. com a TAG <DEL>

Texto inserido

Podemos marcar um texto como inserido, para dar uma ênfase e destacar que ele foi inserido depois. com a TAG <INS>

Texto sublinhado

Podemos usar sublinhado usando a TAG <U> (Não semântica).

Texto sobescrito

Para fazer uma equação, por exemplo: X3 + 2
TAG <SUP>

Texto subescrito

Para fazer uma fórmula química, por exemplo: H20
TAG <SUB>
